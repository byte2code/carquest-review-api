# Changelog

All notable changes to this project are documented in this file.

## [v2.0.0] - 2026-04-18

### Summary
Initial publication of the CarQuest Review Service as a clean, portfolio-ready Spring Boot microservice.

### Highlights

- Added standardized `README.md` with API overview, run steps, and endpoint summary.
- Added `CHANGELOG.md` for future version tracking.
- Cleaned IDE files, temp files, and build artifacts before publishing.
- Preserved the original review CRUD workflow under `/review`.

### Notes

This service is intended to run alongside the CarQuest API (for example on `8081`) as a companion microservice.
